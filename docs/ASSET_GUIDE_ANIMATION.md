# UI演出ブラッシュアップ 素材ガイド

本格的な演出強化(動画・アニメーション)に向けて準備してほしい素材をまとめたもの。
既存の動画ガチャ演出パイプライン(`src/lib/mp4-probe.ts` によるMP4解析、`/admin/gacha-animations` からのアップロード)を前提とした仕様。

## 実装の優先順位

1. 武将登用(ガチャ)演出 ※最優先
2. ホーム画面(入場演出)
3. 国盗り・天下統一達成演出
4. 図鑑・カード獲得演出

素材が全て揃うまで待つ必要はなく、①から順に素材が届き次第実装を進める想定。
動画がない画面はGSAP等によるコード生成アニメーションで代替する。

---

## ① 武将登用(ガチャ)演出

レアリティ(足軽級/武将級/大名級)ごとに動画を分ける。

| 項目 | 推奨スペック |
|---|---|
| 形式 | MP4(H.264 + AACまたは無音)。mov/webm等の他形式は不可 |
| 解像度 | 縦動画 1080×1920(9:16) |
| 長さ | 大名級: 4〜6秒 / 武将級: 2〜3秒 / 足軽級: 1〜2秒 |
| ファイルサイズ | 1本あたり3MB以内が目安 |
| ポスター画像 | 動画の最終フレーム相当の静止画(webp/jpg)を1枚。読み込み中のちらつき防止用 |
| 音声 | 任意。入れる場合もミュート再生で成立する演出にする(自動再生はミュート前提) |
| 本数 | 最低3本(各レアリティ1本)。武将ごとに変える場合は追加可(武将×スロット単位で登録可能) |

## ② ホーム画面(入場演出)

- ループ可能な背景動画(5〜10秒でシームレスループ)、または静止画+パーティクル(桜吹雪・提灯等)をコードで動かす方式でも可
- 動画を使う場合: 1080×1920 / 5MB以内 / 音声なし(常時再生のため無音必須)
- 動画を使わない場合: 高解像度の背景イラスト(和紙・城下町の遠景等)1枚があれば、動きはコード側(GSAP)で付与する

## ③ 国盗り・天下統一達成演出

- お祝い演出用の短尺動画(3〜5秒 / 1080×1920 / 3MB以内)、または家紋・花火・巻物などの透過PNG/SVGイラスト(静止画複数枚でも、コード側でアニメーション化可能)
- 天下統一(最終達成)は他の演出より尺を長めにし、特別感を出す

## ④ 図鑑・カード獲得演出

- 動画は不要。武将ごとの高解像度イラスト(背景透過PNG推奨、既存の武将画像を流用可)があれば、めくり・光エフェクトはコード側で実装する

---

## アップロード方法

動画素材が揃った武将登用演出から、管理画面の `/admin/gacha-animations` 経由でアップロードする。
