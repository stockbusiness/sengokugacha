# 戦国経済圏OS 実装済み機能一覧

最終更新: 2026-07-09

## 1. ユーザー向け機能(LIFF)

### ログイン・基盤
- LINE LIFF連携ログイン。全ページで共通のログイン保証処理(未ログイン時は自動でLIFFログインへ誘導)
- 代理店紐付け: URLパラメータ経由でアクセスしたユーザーの登録時に代理店IDを記録
- 連続ログインボーナス(登城ボーナス)
- 退会(アカウント削除申請)導線

### 戦国パスポート・ガチャ
- 戦国パスポート基本画面(所持武将・進捗などのホーム画面)
- ガチャ抽選ロジック(無料枠)と結果表示画面
- 排出率はDBの `gacha_rate_tiers` を参照して制御(管理画面から変更可能)
- Stripe決済による石高・ガチャ券購入、有料ガチャ
- 購入前の確認ダイアログ、月間利用上限による使いすぎ防止
- **動画ガチャ演出**: レアリティ・新規排出有無に応じた演出動画をフルスクリーン再生(状態遷移・タイムアウト・スキップ・ミュート・タップ再生フォールバック対応)。演出再生の分析イベントをログ記録

### 国盗り・地図・実績
- 国盗り判定ロジック(地方コンプリート・美濃国解放などの特殊条件)
- 日本地図の進捗表示画面
- 地方コンプリート画面と石高ボーナス付与
- 天下統一達成フロー(専用演出画面・代表武将選択・実績記録)

### 図鑑・情報ページ
- 図鑑・所持武将一覧画面
- 遊び方ページ(ゲームの遊び方ハブ。FAQ・お知らせ・お問い合わせへのリンクを集約)
- FAQページ、お知らせページ
- 排出率公開ページ(`/rates`)
- 法的ページ(利用規約・プライバシーポリシー等、`/legal/[slug]`)
- お問い合わせ窓口ページ

### 送客導線
- AIアート教室・NFTマーケット・評議員などの外部送客リンク(管理画面で編集可能)

## 2. 管理画面(Admin)

### マスタ管理
- 国(province)マスタ編集
- 武将(warlord)マスタ編集。画像アップロード(リサイズ・保存)対応
- 排出率設定(`/admin/gacha-rates`)
- ガチャ全体設定(ガチャ券コスト・連続ログインボーナス等)
- 動画演出(gacha-animations)の登録・編集・プレビュー。MP4アップロード時に長さ・解像度・音声有無を自動解析して検証

### LINE連携設定
- LIFF ID・LINEチャネルIDの管理画面からの設定
- LINEリッチメニューの自動作成・配信機能
- リッチメニュー画像の管理画面からのアップロード・差し替え(全体一括、および6ボタンを個別に差し替え→自動再合成の両対応)
- LINE一斉配信(プッシュ通知)機能

### 運用・分析
- 認証つき管理画面(ログイン/ログアウト)
- 操作ログ(監査ログ)記録
- KPIダッシュボード
- 売上ログ・実績ログ・ユーザー検索
- 購入一覧・返金処理
- 代理店報酬の支払い状況管理
- 代理店(agent)マスタ管理
- 外部送客リンク管理
- 法的ページ編集
- FAQ・お知らせ編集
- 決済設定(Stripe等)管理

## 3. 基盤・非機能

- Supabase Postgres + Storage(サービスロールでのサーバー操作、RLS有効)
- 画像アップロード共通基盤(`sharp`によるリサイズ・圧縮、用途別プリセット: 武将画像/リッチメニュー全体/リッチメニューパネル/ガチャ動画ポスター)
- 純JS実装のMP4(ISO BMFF)パーサーによる動画メタデータ解析(ffmpegなどのネイティブ依存なし)
- エラー監視(Sentry)導入
- CI(GitHub Actions)導入
- Stripe Webhook連携
