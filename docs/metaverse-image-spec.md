# 城下町デジタル内覧 画像仕様書

戦国パスポート「戦国城下町デジタル内覧」機能で使用する画像・動画素材の仕様一覧。制作前に必ずこのドキュメントを確認してください。

## 共通仕様(全画像共通)

- アップロード後、サーバー側で自動的に**長辺1080px・WebP形式(quality 85)**にリサイズ・変換される。それ以上の高解像度で用意しても最終表示には反映されないが、トリミング作業のしやすさのため、ある程度の解像度(2000px前後)で用意することを推奨
- アップロード時のファイルサイズ上限: **10MB**(元データ。変換後は自動的に小さくなる)
- 対応形式: JPEG / PNG(どちらでも可。アップロード時に自動でWebPへ変換される)
- 動画のみ例外: MP4形式・50MB以内・60秒以内(変換処理はされないため、あらかじめ適切な解像度・ファイルサイズで書き出すこと)

## 1. 全体マップ画像

| 項目 | 仕様 |
|---|---|
| 用途 | LIFF「エリア一覧」画面上部に表示する、城下町の俯瞰イラスト。タップ可能なエリアの領域(ポリゴン)または点を重ねて表示する |
| **アスペクト比** | **必ず正方形(1:1)** ※重要。理由は下記参照 |
| 推奨解像度 | 2048×2048〜4096×4096px |
| アップロード先 | 管理画面「全体マップ管理」(`/admin/metaverse/maps`) |

**正方形が必須の理由**: システム側の座標系(viewBox)が既定で4096×4096(正方形)に設定されており、管理画面でエリアの範囲を描く作業も、LIFF側での表示も、この座標系を基準に画像へ重ねて表示する。画像が正方形でない場合、管理画面で描いた範囲とLIFF側の表示位置がずれる。**正方形以外のアスペクト比(横長・縦長)で制作したい場合は、事前に必ず知らせてください**。システム側の設定を実際の画像比率に合わせて調整します(現状は正方形のみサポート、比率変更にはコード修正が必要)。

**内容の目安**: 各エリアの境界がある程度視覚的に判別できる俯瞰イラスト(色分け・区画線があると、管理画面でエリアの範囲を描く作業の目安になりやすい)。

## 2. エリア画像

エリア(城下町内の区画のグループ。例: 「創設代理店特区」「一般販売区」等)ごとに2種類の画像を設定できる。

| 項目 | 一覧サムネイル画像 | 詳細ページのメイン画像 |
|---|---|---|
| 用途 | LIFFエリア一覧のカードサムネイル | (現状LIFF側では未使用。将来の拡張用に予約) |
| 表示サイズ | 64×64px(正方形、中央トリミング) | ― |
| 推奨アスペクト比 | 正方形(1:1) | 正方形〜横長(4:3程度まで) |
| 推奨解像度 | 800×800px程度 | 1080px程度(長辺) |
| アップロード先 | 管理画面「エリア管理」(`/admin/metaverse/areas`) |

未設定の場合、後述の「エリアのデフォルト画像」が使われる。

## 3. 物件(区画)画像

| 項目 | メイン画像 | 画像ギャラリー(複数枚) |
|---|---|---|
| 用途 | エリア詳細ページの物件カード / 物件詳細ページのヒーロー画像 / お気に入り一覧 | 管理画面での参考資料用(現状LIFF側の物件詳細ページには表示されない) |
| 表示サイズ | カード: 横幅いっぱい×96px(トリミングあり) / 詳細ヒーロー: 横幅いっぱい・自然なアスペクト比のまま(トリミングなし) / お気に入り: 56×56px正方形 | 管理画面: 96×96px正方形 |
| 推奨アスペクト比 | 横長(4:3〜16:9) | 自由(トリミングされないため見た目のまま保存される) |
| 推奨解像度 | 1080px程度(長辺) | 同上 |
| アップロード先 | 管理画面「区画・物件管理」→ 各物件の編集ページ |

物件詳細ページのヒーロー画像は**トリミングされず自然な比率のまま表示される**ため、極端な縦長・横長は避けること。メイン画像未設定の場合、「物件のデフォルト画像」が使われる。

## 4. 内覧シーン画像・動画

外部内覧ページ(スワイプで切り替わる全画面ビューア)で表示するシーンごとの素材。

| 項目 | シーン画像 | シーンサムネイル | シーン動画(任意) |
|---|---|---|---|
| 用途 | 全画面メイン表示(画面に収まるよう自動調整、トリミングなし) | 画面下部の切替サムネイル一覧 | 設定すると静止画の代わりに再生される |
| 表示サイズ | 画面いっぱい(高さ基準、トリミングなし) | 80×56px(横長) | 画像と同じく画面いっぱい・トリミングなし |
| 推奨アスペクト比 | **横長(16:9〜4:3)を推奨** ※縦長だと画面の左右に大きな余白ができる | 同上 | 横長推奨 |
| 推奨解像度 | 1080px程度(長辺) | 同上(画像から自動生成も可) | 指定なし。60秒以内・50MB以内で書き出すこと |
| 形式 | JPEG/PNG(自動でWebP変換) | 同左 | **MP4のみ**(変換処理なし) |
| アップロード先 | 管理画面「区画・物件管理」→ 各物件の編集ページ → 内覧シーン |

動画を設定した場合も、シーン画像は「動画のポスター画像・フォールバック」として必須のまま残る(削除不可)。

## 5. デフォルト画像(共通フォールバック)

個別に画像を設定していないエリア・物件で使われる、共通の代替画像。

| 項目 | エリアのデフォルト画像 | 物件のデフォルト画像 |
|---|---|---|
| 仕様 | 「2. エリア画像」の一覧サムネイル画像と同じ仕様(正方形推奨) | 「3. 物件画像」のメイン画像と同じ仕様(横長推奨) |
| アップロード先 | 管理画面「エリア管理」ページ下部「デフォルト画像設定」(`/admin/metaverse/areas`) |
