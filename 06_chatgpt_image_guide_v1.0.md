# 戦国経済圏OS ChatGPT画像生成 制作ガイド Ver.1.0

作成日: 2026年7月6日
対象: 武将カードイラスト(75枚)・城下町コンセプトアート(25〜26国分)の制作
使用ツール: ChatGPT(画像生成機能)

---

## 1. 基本方針

- 既に確認できた3枚(金・黒・赤基調、家紋をあしらった構図)の質感を「戦国経済圏OSの公式スタイル」として固定する
- ChatGPTには Midjourney の `--sref`(スタイル参照)のような専用機能はないが、**「この画像と同じ画風で」と指定して参考画像を毎回アップロードする**ことで、同等の効果が得られる
- 生成のたびに、以下の**3点セット**を必ず入力する
  1. 参考画像(既存の完成品、または直前に生成して気に入った1枚)
  2. 共通スタイル指示(下記2章)
  3. 個別の武将/国の特徴(下記3〜4章)

---

## 2. 共通スタイル指示(毎回使うテンプレート文)

以下の文章を、生成のたびに**そのまま冒頭に貼り付ける**。

```
この参考画像と同じアートスタイル・色調・質感で生成してください。
- 金・黒・赤を基調にした重厚な色合い
- 戦国時代の武将・城下町をテーマにした世界観
- 背景に家紋や金箔模様をあしらった装飾的な演出
- 光と影のコントラストを強めにした、シネマティックで高級感のある仕上がり
- 縦長の構図(ポートレート比率)
```

## 3. 武将カード用の個別指示テンプレート

共通スタイル指示に続けて、以下を追加する。

```
[武将の外見の特徴]を持つ日本の戦国武将の肖像を描いてください。
- 鎧・衣装:[鎧の色・装飾の特徴]
- 表情・雰囲気:[レアリティに応じた表情。下記表を参照]
- ポーズ:[例: 刀に手をかけている/腕を組んでいる/前を見据えている]
- 背景:[家紋のモチーフ、または城下町の一部が薄く見える]
```

### レアリティ別の演出差

| スロット | 表情・雰囲気の指定例 |
|---|---|
| コモン枠(足軽級/侍級) | 素朴で落ち着いた表情、装飾は控えめ |
| 中間枠(武将級) | 貫禄のある表情、鎧に金の装飾を追加 |
| レア枠(軍師級/大名級) | 鋭い眼光、躍動感のあるポーズ、背景の家紋・演出を最も豪華に |

### 記入例(レア枠武将)
```

鋭い眼光を持つ日本の戦国武将の肖像を描いてください。
- 鎧・衣装:黒地に金の装飾を施した鎧、赤い襟の羽織
```

## 5. 一貫性を保つコツ(ChatGPT特有の注意点)

- **参考画像は毎回アップロードする**(過去のチャット履歴に頼らず、新しい会話でも同じ参考画像を都度添付する方が安定する)
- 一度に大量生成しようとせず、**1枚ずつ生成→確認→微調整**のサイクルで進める(まとめて指示すると画風がブレやすい)
- 気に入った1枚が出たら、その画像を**新しい参考画像として次の生成に使う**(生成を重ねるごとに、直近の気に入った1枚に差し替えていくと、シリーズ全体の統一感が保ちやすい)
- 武将名(織田信長、豊臣秀吉等)は指示文に入れず、**外見の特徴のみ**を記述する(実在の肖像画や既存作品に寄せた生成を避けるため)

---

## 6. 制作の進め方(推奨フロー)

1. 今回の3枚から**基準となる1枚**を選定(または新たに理想形を1枚生成)
2. その1枚を「公式リファレンス画像」として保存
3. 武将75体分・国25〜26箇所分について、上記テンプレートの[ ]部分だけを差し替えながら、リファレンス画像を都度添付して生成
4. 生成物を選定・保存し、`warlords.image_url` / `provinces.castle_town_concept_art_url` として登録

---

## 7. 注意点

- ChatGPTでの生成にも著作権・肖像権の配慮は必要。実在の人物や既存の版権キャラクターに酷似した結果が出た場合は使用しない
- 大量生成の際、ChャットGPTの利用制限(生成回数の上限等)に達する可能性があるため、契約プランの利用上限を事前に確認しておくと安心
