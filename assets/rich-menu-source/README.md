`richmenu-unification.webp`(旧6枠目「天下統一」)は現在の結合画像では使用していないが、
差し替え候補として保管している。天下統一はパスポート画面・ガチャ画面に個別の導線があるため、
リッチメニューの6枠目は新規ユーザー向けの「遊び方」を優先した(詳細は各PRの説明を参照)。

## 元画像とサイズの都合について

パスポート/ガチャ/図鑑/日本地図/購入の5枚は横長(約2:1)、遊び方は正方形(約1:1、
1254×1254)と素材ごとにサイズが異なる。一方LINEの2行×3列レイアウトの1マスは
約833×843px(ほぼ正方形)。横長素材をそのまま引き伸ばすと歪みや文字切れ・
主役の欠落が発生する。

そのため `public/rich-menu.jpg` では、各マスを「単体画像をぼかして引き伸ばした背景 +
単体画像そのもの(無加工・無トリミング)を中央に配置」という構成にして、
歪みなし・文字切れなし・主役欠落なしで2500×1686pxに収めている。

レイアウトや素材を変更する場合は、この単体画像を差し替えたうえで、
上記と同様の合成処理で結合画像を作り直すこと(単純な等分割トリミングは行わないこと)。
