## 元画像とサイズの都合について

単体画像は横長(約2:1)だが、LINEの2行×3列レイアウトの1マスは約833×843px(ほぼ正方形)。
そのままだと大きく引き伸ばすか、左右を半分近く切り落とさないと収まらず、
歪み・文字切れ・家紋や旗など主役の欠落が発生する。

そのため `public/rich-menu.jpg` では、各マスを「単体画像をぼかして引き伸ばした背景 +
単体画像そのもの(無加工・無トリミング)を中央に配置」という構成にして、
歪みなし・文字切れなし・主役欠落なしで2500×1686pxに収めている。

レイアウトや素材を変更する場合は、この単体画像を差し替えたうえで、
上記と同様の合成処理で結合画像を作り直すこと(単純な等分割トリミングは行わないこと)。
